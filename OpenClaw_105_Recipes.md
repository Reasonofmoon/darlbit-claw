# OpenClaw (Moltbot) 실전 활용 105가지 레시피: 리스트 & 구현 가이드 (v5.0)

## 1. 개요
본 보고서는 105가지의 OpenClaw 활용 사례와 각 사례를 구현하기 위한 **핵심 도구(Skills) 및 논리적 단계**를 요약합니다.

---

## 2. 분야별 105가지 상세 레시피

### 📊 [1-15] 생산성 & 개인 지식 관리 (PKM)
1. **회의록 자동 파싱**: `openai-whisper`로 녹음본 전사 → LLM이 요약 → `obsidian`에 마크다운 저장.
2. **독서 하이라이트 통합**: `web_fetch`로 킨들 클라우드 접속 → 하이라이트 추출 → `bear-notes`로 전송.
3. **Daily Briefing**: `cron`으로 매일 아침 `weather` + `gog calendar` 정보 취합 → `message`로 텔레그램 발송.
4. **캘린더 최적화**: `gog calendar`로 중복 일정 감지 → `himalaya` 스킬로 상대방에게 일정 조정 메일 발송.
5. **학습 플래시카드**: `read`로 공부 노트를 읽음 → LLM이 문답 생성 → `write`로 Anki용 CSV 저장.
6. **AI 리서치 보고서**: `web_search`로 최신 논문 검색 → `summarize`로 핵심 파악 → 통합 리포트 작성.
7. **아이디어 보관소**: 텔레그램 보이스 수신 → `openai-whisper` 변환 → `apple-notes`에 자동 기록.
8. **주간 회고 자동화**: `memory/` 폴더의 대화 로그 분석 → 한 주의 성과와 지표를 리포트로 생성.
9. **프로젝트 트래킹**: `linear-cli`로 작업 상태 조회 → 지연된 티켓을 식별하여 담당자에게 리마인드.
10. **지식 베이스 Q&A**: `memory_search` 기능을 활용해 수천 개의 로컬 메모 중 정답 위치 식별 후 답변.
11. **PDF 데이터 추출**: `nano-pdf`로 문서 텍스트 추출 → 특정 표 데이터를 정규화하여 JSON/CSV 저장.
12. **포모도로 타이머**: `exec`로 타이머 실행 → 시간 종료 시 `openhue` 조명 제어 및 알림 메시지.
13. **일정 시각화**: `gog calendar` 데이터를 분석 → `canvas` 도구에 주간 시간 사용량 차트 렌더링.
14. **기술 문서 자동화**: Git 커밋 감지 → 변경된 함수에 맞게 `edit` 도구로 README.md 예제 코드 수정.
15. **이메일 답장 제안**: `himalaya`로 새 메일 읽기 → 내 과거 답변 스타일 학습 → 3가지 초안 생성.

### 💰 [16-30] 금융, 쇼핑 & 비즈니스
16. **자동차 가격 협상**: `web_search`로 시세 파악 → `message`로 딜러들에게 견적 요청 이메일 대량 발송.
17. **장보기 오토파일럿**: `browser` 도구로 마트 접속 → `act:fill`로 상품 검색 및 장바구니 담기 자동화.
18. **인보이스 자동 발행**: `read`로 업무 로그 확인 → `nano-pdf`로 청구서 생성 → `himalaya`로 전송.
19. **영수증 폴더 분류**: `himalaya`에서 PDF 첨부파일 다운로드 → `exec`의 `mv` 명령으로 카테고리별 이동.
20. **Polymarket 분석**: `web_search`로 실시간 이슈 검색 → 확률 계산 에이전트(`spawn`)가 베팅 전략 보고.
21. **암호화폐 고래 감시**: `web_fetch`로 온체인 데이터 사이트 감시 → 대규모 트랜잭션 발생 시 즉시 알림.
22. **쇼핑몰 최저가 추적**: `browser`로 상품 페이지 정기 방문 → 가격 요소 스크레이핑 → 목표가 비교.
23. **Shopify 재고 관리**: `shopify-api` 스킬로 재고 부족 알림 수신 → 공급처에 발주 메일 자동 초안 작성.
24. **세금 환급 계산기**: `read`로 지출 내역 파일 분석 → 세법 기준 적용 → 예상 환급액 시뮬레이션 보고.
25. **사업 아이디어 검증**: `web_search`로 시장 규모 분석 → 경쟁사 웹사이트 `fetch` 후 SWOT 분석 수행.
26. **배달 음식 상태 추적**: `ordercli` 스킬로 배달 현황 확인 → 도착 5분 전 조명(`openhue`) 깜빡이기.
27. **구독 서비스 관리**: 카드 내역 분석 → 중복되거나 미사용 중인 정기 결제 건을 식별해 리스트업.
28. **주식 포트폴리오 리밸런싱**: 현재 자산 비율 확인 → 목표 비중과 차이 계산 → 매수/매도 수량 제안.
29. **법률 계약서 검토**: `read`로 계약서 탐색 → 독소 조항(책임 소재 등) 키워드 검색 및 법률적 해석 보고.
30. **CRM 자동화**: 대화 내용 요약 → `notion` API 또는 `edit` 명령으로 고객 관리 파일 업데이트.

### 🏠 [31-45] 스마트 홈, IoT & 하드웨어
31. **지능형 난방**: `weather`로 내일 기온 확인 → `exec`로 보일러 API 호출하여 온도 선제적 조절.
32. **3D 프린터 원격 제어**: `bambu-cli`로 프린터 상태 확인 → 카메라 스냅샷 분석 후 이상 시 `kill`.
33. **CCTV 분석**: `camsnap`으로 사진 캡처 → `image` 도구로 인물/물체 식별 → 가족 외 인물 감지 시 알림.
34. **Sonos 멀티룸**: `sonoscli`로 거실, 침실 스피커 그룹화 → 상황별(기상/취침) 플레이리스트 재생.
35. **회의 중 방해 금지**: `gog calendar` 회의 시작 시 → `openhue`로 방문 밖 조명 색상 변경.
36. **스마트 식물 관리**: 토양 센서 데이터(JSON) `read` → 수분 부족 시 사용자에게 물 주기 알림 전송.
37. **미세먼지 알리미**: `web_search`로 현재 대기질 확인 → 나쁨 이상일 때 공기청정기(`exec`) 가동.
38. **Eight Sleep 연동**: `eightctl`로 수면 데이터 분석 → 피로도가 높으면 다음 날 `cron` 알람 시간 늦추기.
39. **택배 도착 Vision**: 현관 카메라 스냅샷 → `image` 도구로 박스 브랜드(쿠팡 등) 식별 후 보고.
40. **에너지 절약 모드**: `cron`으로 심야 시간대 확인 → 대기 전력이 높은 기기(`exec`) 자동 전원 차단.
41. **스마트 조명 페어링**: `video-frames`로 화면 분석 → 평균 색상을 `openhue` 조명에 실시간 반영.
42. **쓰레기 배출 알림**: 캘린더의 분리수거 일정 확인 → 전날 밤 9시에 텔레그램으로 배출 품목 공지.
43. **홈 서버 상태 모니터링**: `exec`로 `top` 명령 실행 → CPU/메모리 임계치 초과 시 경고 및 조치.
44. **반려동물 자동 급식**: `cron` 시간에 맞춰 급식기 명령 전송 → 잘 먹는지 카메라로 확인 후 사진 전송.
45. **스마트 거울 연동**: `canvas`에 오늘의 브리핑 화면 렌더링 → 스마트 거울용 디스플레이에 출력.
